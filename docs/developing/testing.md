# Testing

To run the test suite after installing Ocean4DVarNet : 

Install dev dependencies (pylint, pytest, ...) in the models directory of the Ocean4DVarNet repository.
``` bash
pip install .[dev]
```

And the run :

``` bash
pylint

pytest
```
